📋 Cybersalt MCP for Joomla — Changelog

v1.0.0 2026-04-25

Initial release with 51 built-in tools across 11 Joomla admin domains.

📦New Features

  • MCP endpoint: Streamable-HTTP MCP server at /api/index.php/v1/mcp, authenticated by Joomla API token (X-Joomla-Token or Authorization: Bearer).
  • JSON-RPC 2.0 server: initialize, notifications/initialized, ping, tools/list, tools/call. Single messages and batches supported.
  • ACL gating: csmcpforj.use (read) and csmcpforj.write (write) actions. Super Users / Administrator / Manager pass automatically.
  • Tool registry: Plugins extend the surface by subscribing to the onCsMcpRegisterTools event.
  • Admin dashboard: Endpoint URL, ready-to-paste client config, permissions table, and grouped tool list.
  • Built-in tool set (51 tools): Articles, Categories (any extension), Tags, Menus, Users & Access, Modules, Extensions/Plugins, Templates, Languages, Custom Fields, System.

🔍Security

  • Per-tool permission gate runs before every tools/call execution.
  • access.xml ships with the component so per-action permissions appear in System → Permissions immediately after install.
  • All writes route through the corresponding com_* Administrator model so workflow, asset, and ACL side effects stay consistent.
  • Database access uses quoteName() and parameterised values throughout — no string concatenation into SQL.
  • delete_user refuses to delete the calling user.
  • set_extension_enabled refuses to touch protected or locked core extensions.
  • get_site_info deliberately omits secrets (DB password, mailer credentials, captcha keys).
